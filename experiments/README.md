# Experiments Folder

This folder contains experimental code, prototypes, and scratch work.

**This folder is gitignored** - contents are not tracked in version control.

## What goes here:

- Quick prototypes
- Performance benchmarks
- Spike solutions
- Throwaway test scripts
- Local experiments

## Guidelines:

- Don't expect code here to be maintained
- Feel free to delete anything
- No code review required
- Use subdirectories to organize

## Examples:

```
experiments/
  ├── llm_benchmarks/
  ├── workflow_prototypes/
  ├── tool_experiments/
  └── scratch/
```
